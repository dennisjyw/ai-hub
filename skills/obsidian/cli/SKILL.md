# Obsidian CLI

## 何時使用
- 要透過 CLI 操作 Obsidian vault。
- 搜尋、建立、搬移、更新 notes 與 properties。

## 執行流程
1. 先確認目標 vault 與路徑。
2. 先判斷是讀取、搜尋、建立、更新還是批次操作。
3. 只使用與任務直接相關的指令，不做多餘掃描。
4. 若涉及大量變更，先說明影響範圍再動手。

## 產出要求
- 回報實際操作的檔案與命令意圖。
- 若 CLI 無法滿足需求，明確改走 Markdown 或 Canvas 技能。
