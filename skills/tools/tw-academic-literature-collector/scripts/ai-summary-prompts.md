# AI 智能摘要 Prompt 範本

## 單篇文獻摘要生成

### 標準摘要 (500字)

```
請針對以下繁體中文學術文獻生成結構化摘要:

標題: {title}
作者: {authors}
年份: {year}
摘要原文: {abstract}
期刊: {source}

請以IMRAD格式產出500字的結構化摘要:

## 1. 研究背景與問題 (100字)
- 研究動機
- 研究問題或假設

## 2. 研究方法 (150字)
- 研究設計 (明確標註: 質性/量化/混合)
- 研究對象與樣本
- 料蒐集方法
- 分析方法

## 3. 主要研究發現 (150字)
- 核心發現 (條列3-5點)
- 重要數據或證據

## 4. 討論與啟示 (100字)
- 理論貢獻
- 實務意涵
- 研究限制
- 未來研究方向

要求: 使用繁體中文,學術語言但保持可讀性
```

### 短摘要 (300字)

```
請用300字總結這篇文獻,包含研究問題、方法和主要發現。
```

### 超短摘要 (100字)

```
請用100字以內總結這篇文獻的核心發現。
```

## 主題式跨文獻整合摘要

```
請分析以下{n}篇關於「{主題}」的繁體中文文獻:

{文獻清單與摘要}

請產出800-1000字的主題式整合摘要:

## 1. 主題概述 (100字)
- 說明該主題在研究領域中的重要性

## 2. 共識發現 (300字)
- 多篇文獻一致支持的核心觀點
- 引用具體文獻證據,格式: (作者,年份)

## 3. 爭議與差異 (250字)
- 文獻間的不同立場或矛盾發現
- 分析可能原因

## 4. 方法論趨勢 (150字)
- 該主題常用的研究方法統計
- 方法學優缺點評析

## 5. 研究缺口與未來方向 (200字)
- 目前研究尚未充分探討的面向
- 具體的未來研究問題 (3-5個)
```

## 五維度研究缺口分析

```
請進行「{研究主題}」領域的系統性研究缺口分析:

## 資料庫概況
- 總文獻數: {total_count}篇
- 時間範圍: {year_range}
- 研究方法分布: {method_stats}
- 研究情境分布: {context_stats}

請從以下五個維度進行深度分析:

### 1. 理論缺口
- 已被充分探討的理論視角?
- 尚未被應用或不足的理論視角?
- 跨理論整合機會?

### 2. 方法缺口
- 質性/量化/混合方法比例是否平衡?
- 缺乏的特定研究設計?
- 可應用的創新方法?

### 3. 情境缺口
- 已研究的地理/組織/技術情境?
- 尚未研究的情境空白?
- 台灣本土情境研究是否充足?

### 4. 對象缺口
- 主要研究對象統計?
- 被忽略的重要族群?
- 樣本多元性是否足夠?

### 5. 時間缺口
- 研究年份分布是否均衡?
- 是否有研究斷層?
- 需要長期追蹤研究?

最後提出 8-10個具體的未來研究方向建議,
每個包含:研究問題、建議方法、預期貢獻、實施難度評估。
```
