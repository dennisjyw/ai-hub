# 繁體中文學術文獻蒐集與整理系統
**Taiwan Academic Literature Collector & Organizer**
**v2.0 - 新增視覺化網絡圖與AI智能摘要功能**

## 📖 簡介

這是一個專為台灣學術研究者設計的文獻蒐集與整理技能,能夠協助使用者:
- 系統化搜尋台灣學術資料庫的繁體中文文獻
- 自動進行文獻品質評估與分級
- 生成結構化的 Excel 文獻整理表
- 撰寫符合 APA 7th 格式的 Word 文獻綜述
- **🆕 產出互動式文獻視覺化網絡圖**
- **🆕 使用AI生成智能文獻摘要與缺口分析**

## 🎯 核心功能

### 1. 多資料庫搜尋策略
- **Google Scholar** - 直接搜尋繁體中文文獻
- **碩博士論文網** - 提供搜尋策略與指引
- **華藝線上圖書館** - 提供搜尋策略與指引  
- **TCI-HSS** - 提供搜尋策略與指引

### 2. 智能文獻評估
根據以下指標進行文獻品質分級:
- 期刊等級 (TSSCI/THCI/一般)
- 引用次數
- 發表年份
- 研究方法
- 主題相關性

### 3. 專業輸出格式
- **Excel 文獻整理表** - 包含7個工作表 🆕
  - 文獻總覽
  - 文獻摘要
  - APA 引文清單
  - 研究方法統計
  - 主題分類
  - **🆕 AI 智能摘要**
  - **🆕 主題整合分析**
  
- **Word 文獻綜述** - 包含完整章節
  - 文獻綜述概述
  - 主題分類綜述
  - 研究趨勢與缺口分析
  - **🆕 AI 智能分析報告**(含五維度缺口分析)
  - APA 7th 格式參考文獻

### 4. 🆕 互動式視覺化網絡圖
- **引用網絡圖** - D3.js 力導向布局,展現文獻引用關係
  - 節點大小依引用次數調整
  - 節點顏色依年份漸層
  - 互動功能:懸停/點擊/拖曳/篩選
  
- **研究趨勢時間軸** - Recharts 圖表
  - 文獻數量年度趨勢
  - 研究方法演變分析
  - 累積引用趨勢
  
- **關鍵字共現網絡** - TF-IDF + 網絡分析
  - 自動萃取高頻關鍵字
  - 社群偵測識別主題群組
  - 視覺化研究主題分布
  
- **作者合作網絡** - 學術合作關係圖
  - 識別核心作者
  - 展現合作強度
  - 中心性分析

### 5. 🆕 AI 智能文獻分析
- **單篇文獻摘要** - IMRAD格式結構化摘要
  - 超短(100字)/短(300字)/標準(500字)/詳細(1000字)
  - 自動識別研究方法(質性/量化/混合)
  - 條列核心發現
  
- **主題式整合摘要** - 跨文獻綜合分析
  - 共識觀點整理
  - 爭議觀點討論
  - 方法論趨勢分析
  
- **研究缺口系統分析** - 五維度深度分析
  - 理論缺口:未探討的理論視角
  - 方法缺口:質量方法平衡性
  - 情境缺口:地理/組織/技術情境
  - 對象缺口:被忽略的研究群體
  - 時間缺口:研究時序分析
  
- **未來研究建議** - AI輔助生成
  - 8-10個具體研究問題
  - 建議研究方法與設計
  - 預期學術貢獻評估
  - 實施難度分析

## 🚀 使用方式

### 方式一:直接搜尋(含完整分析)
```
使用者: 請幫我蒐集關於『AI在高等教育應用』的台灣繁體中文文獻,
       時間範圍2020-2025,需要包含期刊論文和碩博士論文。
       請產出完整的視覺化分析和AI智能摘要。
```

### 方式二:上傳檔案整理(含視覺化)
```
使用者: [上傳從資料庫下載的 Excel/CSV 檔案]
       幫我整理這些文獻,產生綜述、視覺化網絡圖和AI分析報告。
```

### 方式三:特定資料庫查詢
```
使用者: 查找 TSSCI 期刊中關於『公民參與』的文獻,
       最近5年的研究。
```

### 方式四:專注視覺化分析 🆕
```
使用者: 我有20篇文獻,請幫我產生引用網絡圖、關鍵字共現圖,
       以及研究趨勢分析。
```

### 方式五:專注AI智能分析 🆕
```
使用者: 請用AI分析這些文獻的研究缺口,並提出未來研究方向建議。
```

- 自動處理上傳的資料庫匯出檔案
- 智能識別和標準化不同資料庫的欄位格式
- **🆕 使用 D3.js 建立互動式引用網絡圖**
- **🆕 使用 Recharts 生成研究趨勢視覺化**
- **🆕 使用 TF-IDF 演算法進行關鍵字萃取**
- **🆕 使用 Louvain 社群偵測識別研究主題**
- **🆕 使用 AI 生成 IMRAD 格式結構化摘要**
- **🆕 使用 AI 進行五維度研究缺口系統分析**

## 🧪 測試案例

技能包含 **5 個完整的評估測試案例**:

1. **eval-1-education-ai**: 教育科技主題的完整文獻蒐集流程(含視覺化和AI摘要) 🆕
   - 22項期望檢查點
   - 測試完整工作流程
   
2. **eval-2-uploaded-file**: 處理使用者上傳的文獻清單檔案
   - 11項期望檢查點
   - 測試檔案解析和整合
   
3. **eval-3-tssci-journal**: 特定 TSSCI 期刊文獻查詢
   - 9項期望檢查點
   - 測試期刊等級識別
   
4. **eval-4-multi-database**: 多資料庫整合搜尋策略
   - 8項期望檢查點
   - 測試搜尋策略生成
   
5. **eval-5-visualization-ai**: 專門測試視覺化網絡圖和AI智能摘要功能 🆕
   - 26項期望檢查點
   - 深度測試新功能
