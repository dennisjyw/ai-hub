# 範例對話

## 範例 1: 完整流程

**使用者：**
```
我需要寫一篇關於「數位學習成效評估」的文獻探討,
請幫我蒐集相關的台灣研究,並產出視覺化分析圖和AI智能摘要。
```

**回應流程：**
1. 詢問時間範圍、文獻類型、預期數量、關注子主題
2. 執行 Phase 1-8 完整工作流程
3. 生成 Excel 7工作表檔案
4. 生成 Word 4章節文件
5. 建立視覺化 React Artifacts
6. 執行 AI 缺口分析
7. 使用 present_files 呈現所有檔案

## 範例 2: 檔案整理

**使用者：**
```
[上傳 literature_export.xlsx]
這是我從華藝下載的50篇文獻,幫我整理成文獻回顧。
```

**回應流程：**
1. 解析華藝匯出的文獻資料
2. 補充 Google Scholar 引用數和期刊等級
3. 進行品質評估與主題分類
4. 建立視覺化分析
5. 生成 AI 摘要和缺口報告
6. 呈現所有最終檔案

## 範例 3: 專注視覺化分析

**使用者：**
```
我已經有30篇文獻資料,請幫我產生互動式的引用網絡圖。
```

**回應流程：**
1. 建立 4 個互動式 React Artifacts
2. 整合成單一 HTML 視覺化檔案
3. 生成靜態 PNG 圖檔
4. 使用 present_files 呈現檔案

## 範例 4: 專注 AI 智能分析

**使用者：**
```
我有20篇關於遠距教學的文獻,請用AI幫我分析研究缺口。
```

**回應流程：**
1. 為每篇文獻生成 IMRAD 格式摘要
2. 識別主要研究主題
3. 執行五維度缺口分析
4. 生成具體未來研究建議
5. 更新 Excel 和 Word 檔案
