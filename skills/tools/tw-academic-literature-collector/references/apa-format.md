# APA 7th 中文引用格式

## 中文期刊論文

格式：
```
作者(年份)。標題。期刊名稱,卷(期),頁碼。DOI
```

範例：
```
張三、李四(2023)。人工智慧在教育領域的應用研究。教育科學研究期刊,68(2),45-78。https://doi.org/10.xxxx
```

## 中文學位論文

格式：
```
作者(年份)。標題〔學位論文,大學〕。URL
```

範例：
```
王小明(2022)。台灣高等教育AI應用研究〔博士論文,國立台灣大學〕。https://hdl.handle.net/xxxxx
```

## 內文引用格式

```python
單一作者: (張三, 2023)
兩位作者: (張三、李四, 2023)
三位以上: (張三等, 2023)
多篇同時引用: (李四, 2020; 王五, 2022; 張三, 2023)
```
