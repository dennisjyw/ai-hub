# 測試案例

## 測試案例 1: 教育科技主題

**使用者輸入：**
```
請幫我蒐集關於『AI在高等教育應用』的台灣繁體中文文獻,
時間範圍2018-2025,需要包含期刊論文和碩博士論文。
```

**預期輸出：**
1. Google Scholar 搜尋結果 (20-30筆)
2. 碩博士論文網搜尋指引
3. 華藝資料庫搜尋指引
4. Excel 文獻整理表 (多工作表)
5. Word 文獻綜述 (3000-5000字)

## 測試案例 2: 使用者上傳檔案

**使用者動作：**
上傳從華藝下載的 Excel 檔案 (50筆文獻)

**使用者輸入：**
```
幫我整理這些文獻,產生綜述
```

**預期輸出：**
1. 解析Excel檔案
2. 補充 Google Scholar 引用數
3. 進行品質評估與分級
4. 生成整合後的 Excel 文獻整理表
5. 生成 Word 文獻綜述

## 測試案例 3: 特定期刊查詢

**使用者輸入：**
```
查找 TSSCI 期刊中關於『公民參與』的文獻,
最近5年的研究
```

**預期輸出：**
1. TCI-HSS 資料庫搜尋指引 (標註 TSSCI 篩選)
2. Google Scholar 搜尋結果
3. 期刊等級標註 Excel 表
4. 按期刊分類的 Word 綜述
