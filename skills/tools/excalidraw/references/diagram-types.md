# 圖表類型詳細規範

## 流程圖 (Flowchart)

**結構要素**：
- 開始/結束：橢圓形
- 處理步驟：矩形
- 決策點：菱形
- 流程方向：箭頭

**佈局**：
- 垂直流程：從上到下
- 水平間距：200px
- 垂直間距：120px

## 關係圖 (Relationship)

**結構要素**：
- 實體：矩形或圓角矩形
- 關係線：箭頭或直線
- 關係標籤：文字

**佈局**：
- 網格佈局：sqrt(n) 列
- 元素間距：250-300px

## 心智圖 (Mind Map)

**結構要素**：
- 中心主題：圓形或橢圓（黃色）
- 主分支：4-6 個
- 子主題：每分支 2-4 個

**佈局**：
- 放射狀佈局
- 半徑：150px（第一層）、250px（第二層）

## 架構圖 (Architecture)

**結構要素**：
- 系統層：水平分層
- 元件：矩形
- 連線：箭頭
- 資料流：標註箭頭

**佈局**：
- 分層佈局
- 層間距：150px
- 同層間距：200px

## 時序圖 (Sequence)

**結構要素**：
- 物件：頂部矩形
- 生命線：虛線
- 訊息：水平箭頭
- 啟動框：生命線上的矩形

**佈局**：
- 物件水平排列
- 時間從上到下
- 物件間距：200px
