# 多語言處理原則

## 中文文獻

- 優先提供繁體中文格式
- APA 7th 依台灣學術規範（教育部 / 科技部格式要求）
- 引用格式範例：
  ```
  張三、李四(2023)。標題。期刊名稱,卷(期),頁碼。
  ```

## 英文文獻

- 標準 APA 7th edition
- 引用格式範例：
  ```
  Author, A. A., & Author, B. B. (2023). Title. Journal Name, Volume(Issue), pages.
  ```

## 智庫報告

- 依各機構引用格式慣例
- 同時提供 APA 7th 對應格式

## 混用情境

- 根據段落語言自動切換
- 引用格式與段落語言一致

## 日韓文獻

- 提供羅馬拼音轉寫
- 附原文對照

## 格式對照表

| 情境 | 引用格式 |
|------|---------|
| 中文單一作者 | (張三, 2023) |
| 中文兩位作者 | (張三、李四, 2023) |
| 中文三位以上 | (張三等, 2023) |
| 英文單一作者 | (Author, 2023) |
| 英文兩位作者 | (Author & Author, 2023) |
| 英文三位以上 | (Author et al., 2023) |
| 多篇同時引用 | (李四, 2020; 王五, 2022; 張三, 2023) |
