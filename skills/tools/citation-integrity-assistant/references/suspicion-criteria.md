# 可疑判斷標準

## 判斷為「⚠️ 可疑」的條件

### 作者姓名問題
- 中文轉拼音錯誤
- 西文拼寫錯誤
- 姓名格式異常

### 年份問題
- 年份與作者發表記錄不符
- 年份與期刊出版年份矛盾

### 期刊問題
- 期刊名稱不存在
- 期刊名稱拼寫錯誤
- 卷期號與年份不符

### 標題問題
- 標題查無相符結果
- 標題與摘要內容不符

### 引用格式問題
- 引用格式與內容資訊互相矛盾
- DOI 無法解析

## AI 幻覺特徵

| 特徵 | 說明 |
|------|------|
| 過於完整 | 文獻資訊過於完整、格式完美但查無來源 |
| 資訊矛盾 | 作者、年份、期刊資訊互相矛盾 |
| 查無來源 | 在 Google Scholar、資料庫都找不到 |
| DOI 異常 | DOI 格式正確但無法連結 |

## 智庫報告可疑特徵

- 報告編號與機構風格不符
- 系列名稱不存在
- 作者非該機構研究員
- 發布日期與報告編號矛盾

## 判斷流程

```
1. 解析文獻資訊
   ↓
2. 模型知識驗證（🔵）
   ↓
3. 網路搜尋驗證（🌐）
   ↓
4. 交叉比對
   ↓
5. 判定：✅ 存在 / ⚠️ 可疑 / ❌ 不存在
```
