# 輸出格式範例

## 模組一：反查文獻真實性

### 完整模式輸出

```
【文獻 N】
原始引用：[使用者貼入的原文]
驗證結果：✅ 存在 / ⚠️ 可疑 / ❌ 不存在
信心來源：🔵 知識庫 / 🌐 即時查驗 / 🔵+🌐 雙重確認
問題說明：[說明為何可疑或不存在]
建議替代：[若判斷為可疑或不存在，提供替代文獻或修正建議]
APA 7th 格式：[正確格式]
```

### 精簡模式輸出

```
| # | 原始引用摘要 | 結果 | 問題 |
|---|------------|------|------|
| 1 | Author (YYYY) Title... | ✅ 存在 | — |
| 2 | Author (YYYY) Title... | ⚠️ 可疑 | 期刊名稱無法確認 |
| 3 | Author (YYYY) Title... | ❌ 不存在 | 查無此文獻，疑似 AI 幻覺 |
```

### 整體風險摘要

```
📊 整體風險摘要
- 文獻總數：N 筆
- ✅ 確認存在：N 筆
- ⚠️ 可疑（需人工確認）：N 筆
- ❌ 高度可疑 / 不存在：N 筆
- 🤖 AI 幻覺風險評估：低 / 中 / 高
- 建議行動：[整體建議]
```

## 模組二：段落文獻補充

```
📌 段落主旨：[一句話摘要]
📚 推薦補充文獻

【選項 A】— 強支撐（直接對應核心主張）
作者, 年份. 標題. 期刊 / 出版社.
信心度：🔵🌐
內容說明：[為何適合這段落]
APA 7th：[完整格式]

【選項 B】— 側面佐證（提供背景或對比觀點）
[同上格式]

【選項 C】— 經典基礎文獻（理論奠基）
[同上格式]

💡 引用建議：[建議在段落哪個位置插入引用]
```

---

## 模組三：理論文獻源頭追蹤

```
🔍 理論溯源：[理論 / 概念名稱]

📍 原始來源
作者：[最早提出者]
年份：[提出年份]
原典：[原始著作 / 論文]
APA 7th：[正確格式]
信心來源：🔵 / 🌐 / 🔵+🌐

📖 理論發展脈絡
[簡述理論如何發展]

⚠️ 常見二手引用陷阱
[說明哪些常見引用實際上是轉引]

✅ 建議引用方式
直接引用原典：[格式]
若確實未能取得原典，轉引格式：[as cited in 格式]
```

## 模組四：多文獻引用選擇器

```
📂 概念：[概念名稱]
使用語境：[使用者描述的段落用途]

─────────────────────────────
🏛️ 選項 1｜經典奠基型
[文獻資訊 + APA 7th + 適用場合說明]

📊 選項 2｜近期實證型（YYYY）
[文獻資訊 + APA 7th + 適用場合說明]

🔄 選項 3｜批判 / 修正觀點型
[文獻資訊 + APA 7th + 適用場合說明]

🌏 選項 4｜台灣 / 亞洲情境型（如有）
[文獻資訊 + APA 7th + 適用場合說明]
─────────────────────────────

💬 選擇建議
[根據使用者的段落語境，建議優先使用哪個選項]
```
