# n8n AI Agent 工作流設計提示詞框架

這套框架主要是為了解決一個痛點：很多人想用 n8n 做自動化，但不知道怎麼把自己的想法變成實際的工作流。
你可能遇到過這種情況 - 比如你想自動化處理郵件，或者定時生成報表，想法很清楚，但打開 n8n 後就懵了：我該用哪些節點？怎麼連接？參數怎麼設置？這些技術細節往往比寫代碼還讓人頭疼。
所以我們做了這套框架，讓 AI 來充當你的工作流設計師。你只需要用自然語言描述你想要什麼，比如"我希望每天自動從郵箱下載附件然後上傳到網盤"，AI 就會幫你分析需求，設計出完整的節點架構，甚至生成可以直接導入 n8n 的 JSON 文件。
框架的核心是基於 AI Agent 的設計思路，這樣工作流不只是簡單的數據傳遞，還能做智能判斷和複雜處理。整個過程分三步：先聊需求、再設計架構、最後生成代碼。AI 會搜索 n8n 的官方文檔和社區方案，確保給你的是最佳實踐。
用這套框架，你從想法到可用的工作流，大概 15 分鐘就能搞定。特別適合那些有自動化想法但不知道怎麼在 n8n 裡實現的人，讓 AI 來幫你填補從需求到技術實現之間的落差。

# 核心功能模塊

## 需求分析
智能需求建模 - AI 會通過自然語言對話引導你描述想要自動化的工作場景，然後自動識別觸發條件、數據來源、處理邏輯和輸出目標。AI 會確認你的 n8n 環境版本和相關工具配置，基於這些信息設計出完整的 AI Agent 架構方案，包括節點清單、數據流向和複雜度評估，為後續構建提供清晰的技術藍圖。

## 架構設計
AI Agent 工作流設計 - AI 會自動搜索 n8n 官方文檔、社區方案和模板庫，找到最適合你需求的解決方案。然後為每個節點設計詳細配置，包括 AI Agent 的提示詞設計、DeepSeek v3 模型參數、工具節點的 API 集成和數據轉換邏輯。所有配置都遵循 n8n 的最佳實踐，確保工作流穩定可靠且易於維護。

## 構建
JSON 工作流生成 - 將設計好的架構轉換成符合 n8n 導入標準的 JSON 文件，包含所有節點配置、連接關係和參數設置。生成的工作流以 AI Agent 為核心，確保數據流向正確、節點 ID 唯一，可以直接導入到 n8n 平臺。

## 流程圖
可視化架構展示 - 使用 Mermaid 圖表動態生成工作流的可視化架構圖，清晰展示各節點間的數據流向和執行順序。幫助用戶直觀理解 AI Agent 如何協調各個工具節點，以及整個自動化流程的運行邏輯。

## 狀態
項目進度管理 - 實時分析當前工作流項目的完成情況，包括需求分析完成度、架構設計進展和 JSON 構建狀態。AI 會根據當前階段提供具體的下一步操作建議，確保整個設計流程高效有序地推進。

> 注：本框架專為 n8n AI Agent 工作流設計，特別適合想要構建智能自動化流程但不知道如何設計架構的用戶。最終工作流的複雜度和功能將根據你的具體需求確定，生成的 JSON 文件可直接在 n8n 平臺使用。
