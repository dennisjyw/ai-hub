# shadcn/ui

## 何時使用
- 使用者提到 `shadcn`、`shadcn/ui`、component registry、theme、form、chart。
- 要新增或調整 shadcn 元件，而不是自製整套 UI library。

## 載入策略
- 安裝與初始化：讀 [references/setup-and-configuration.md](references/setup-and-configuration.md)
- 元件選型：讀 [references/ui-components.md](references/ui-components.md) 或 [references/official-ui-reference.md](references/official-ui-reference.md)
- 表單：讀 [references/forms-and-validation.md](references/forms-and-validation.md)
- 圖表：讀 [references/chart.md](references/chart.md) 或 [references/charts-components.md](references/charts-components.md)
- Next.js：讀 [references/nextjs-integration.md](references/nextjs-integration.md)

## 執行流程
1. 先確認專案是否已完成 shadcn 初始化。
2. 選用現成元件與 pattern，避免自己重造輪子。
3. 只補讀與當前任務直接相關的 reference。
4. 交付時附上安裝步驟、變更檔案與 theme／variant 決策。
